## Projeto
Esse é um projeto Web Responsivo de uma cápsula do tempo para exibir memórias numa timeline.

## Tecnologias
Esse projeto foi desenvolvido durante NLW da Rocketseat com as siguintes tecnologias:

- HTML
- CSS 
- Git e Github
